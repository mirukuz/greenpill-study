# Ep.46 · Filecoin：去中心化存储 —— 与 Alan Ransil 对谈

**发布日期**：2022-09-20 · **来源**：Green Pill Podcast

---

## 核心内容

Alan Ransil 是 Filecoin 基金会的成员，探讨去中心化存储如何成为 Web3 和公共物品生态的基础设施。

**主要议题：** Filecoin 的核心价值：让数据存储去中心化，不依赖 AWS、Google Cloud 等单一公司；数据存储作为公共物品：互联网历史、学术数据、开源代码的永久保存；Filecoin 基金会如何资助公共数据存储项目；存储市场的经济设计；与 IPFS（InterPlanetary File System）的关系：IPFS 提供内容寻址，Filecoin 提供存储激励。

**关键洞见：** 数据的永久性问题——放在中心化服务器上的数据会随着公司倒闭或政策变化而消失。Filecoin 提供了一种激励存储节点长期保存数据的机制，对于需要永久保存的重要文档（历史记录、法律文件）意义重大。

---

## 墨尔本本地 LGBTQ+ 社群的启发

LGBTQ+ 历史记录的保存是一个真实的痛点——很多重要的社运历史、个人证言、事件记录因为依赖商业平台而面临消失风险。用 Filecoin/IPFS 存储 LGBTQ+ 历史档案，可以确保这些记录不会因为平台倒闭或内容审查而消失。这是 Web3 工具对 LGBTQ+ 社群最直接的实用价值之一，值得作为早期应用场景优先考虑。
