# Ep.28 · Balaji Srinivasan：集体行动与网络国家

**发布日期**：2022-07-19 · **来源**：Green Pill Podcast

---

## 核心内容

Balaji Srinivasan 是著名的技术投资者和《网络国家》（The Network State）的作者。本集探讨他对"网络国家"的愿景——一个先在网络上形成集体意识，再逐步获得实体领土和外交承认的新型国家形态。

**主要议题：** 网络国家的概念：从"先网络后领土"到"先有一千万认同者再寻求领土"；为什么现有民族国家体系无法满足多元价值观的协调需求；集体行动问题在网络国家中的解法；加密货币作为网络国家的经济基础；Balaji 对 Web3 乐观主义的论证；网络国家与 GreenPill 愿景的连接点和分歧点。

**关键洞见（批判性视角）：** 网络国家理论吸引人但需要批判性审视——它可能被有特权者用来逃避现有社会责任，而非真正解决协调失败。Kevin Owocki 在对话中提供了再生加密角度的补充：好的协调技术应该让现有世界变得更好，而不只是为精英提供退出选项。

---

## 墨尔本本地 LGBTQ+ 社群的启发

网络国家的概念对 LGBTQ+ 社群有吸引力（跨越地理边界的价值观共同体），但也有风险：如果只有有技术能力和经济资源的 LGBTQ+ 成员能参与"网络国家"，则会加剧社群内部不平等。更务实的路径：把网络国家的工具和思想用于增强已有的物理社群，而不是用"网络退出"代替在现实中持续倡导权利改变。
