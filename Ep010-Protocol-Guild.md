# Ep.10 · Protocol Guild —— 与 Trent Van Epps 对谈

**发布日期**：2022-04-21 · **来源**：Green Pill Podcast

---

## 核心内容

Protocol Guild 是以太坊核心开发者的集体资助实验——通过智能合约将协议费用的一部分自动分配给所有活跃的以太坊核心开发者，解决"开源基础设施贡献者缺乏稳定收入"的问题。

**主要议题：** Protocol Guild 的设计：一个链上合约，周期性地将资金按贡献年限和时间加权分配给数百位核心开发者；成员资格如何确定：活跃于以太坊核心协议开发（共识层、执行层、研究）的贡献者；不同项目方（Optimism、Uniswap 等）如何向 Protocol Guild 捐款；这个模式对整个 Web3 开源生态的启示；公共基础设施维护者如何获得可持续收入。

**关键洞见：** Protocol Guild 证明了一件重要的事：开源贡献者不必在使命感和生活保障之间选择。通过链上自动分配机制，可以让数百人的贡献得到可持续认可，而不需要复杂的薪酬谈判。

---

## 墨尔本本地 LGBTQ+ 社群的启发

Protocol Guild 的模式对 LGBTQ+ 社群有直接参考价值：如果社群代币或 NFT 收入有持续来源，可以用类似 Protocol Guild 的智能合约自动、按比例分配给核心贡献者——不需要每次都重新讨论"谁应该得到多少"，链上规则透明且不可撤销，减少了大量人际摩擦。
