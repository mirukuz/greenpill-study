# Ep.158 · Allo Protocol V2：资助重要的事 —— 与 Nate Gosselin 和 Zachary Fleischmann 对谈

**发布日期**：2023-10-20 · **来源**：Green Pill Podcast

---

## 核心内容

Allo Protocol 是 Gitcoin 的底层资金分配协议，V2 版本是重大升级——将协议设计得更模块化、更可组合，任何人都可以在其上构建自定义的资助机制。

**主要议题：** Allo Protocol V2 的技术改进：模块化设计、更低 Gas 费、更好的可组合性；"资助即协议"：将资助机制从中心化平台转变为开放标准；任何人都可以在 Allo 上构建的资助策略（二次方资助、直接资助、追溯性资助）；Allo 生态中涌现的创新应用；对资助生态多元化的影响。

**关键洞见：** Allo Protocol V2 是公共资助基础设施的重要飞跃——它将资助机制从"必须用 Gitcoin 的方式"变成"可以用任何方式，只需要遵循开放标准"。这大幅降低了资助创新的门槛。

## 墨尔本本地 LGBTQ+ 社群的启发

Allo Protocol V2 的开放标准意味着未来可能出现专为 LGBTQ+ 社群设计的资助策略——比如给历史上被边缘化群体（跨性别者、残障酷儿）额外权重的 QF 变体，或者专门追踪 LGBTQ+ 服务影响力的里程碑资助策略。关注 Allo 生态的开发者社区，或直接参与贡献一个 LGBTQ+ 专属的资助策略模块，可能是社群影响整个 Web3 资助生态的高杠杆机会。
